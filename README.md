# موسوعة الجرائم بحق المسلمين عبر التاريخ
# Encyclopedia of Crimes & Massacres Against Muslims

A comprehensive, multilingual historical documentation website of recorded crimes and massacres committed against Muslims worldwide.

## ✨ Features

- **28+ documented events** spanning 1948–present
- **Bilingual** – Arabic (primary, RTL) and English
- **Dark / Light theme** toggle with localStorage persistence
- **Filter by region**: Middle East, Balkans, South Asia, Southeast Asia, Central Asia, Africa, Caucasus, Europe
- **Sort** by date (newest/oldest), victim count, or alphabetical
- **Search** by event name, location, or year
- **Grid / List** view toggle
- **Inline expandable cards** + full-detail modal
- **Verified sources** (Wikipedia, HRW, Amnesty International, UN, ICJ) linked after every event
- Zero dependencies — pure HTML + CSS + JavaScript, single file
- GitHub Pages ready

## 🚀 Deploy to GitHub Pages

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://<username>.github.io/<repo>/`

## 📚 Data Sources

All events are sourced from internationally recognized references:
- [Wikipedia](https://en.wikipedia.org)
- [Human Rights Watch](https://www.hrw.org)
- [Amnesty International](https://www.amnesty.org)
- [United Nations OHCHR](https://www.ohchr.org)
- [International Court of Justice](https://www.icj-cij.org)
- [ICTY – International Criminal Tribunal for the former Yugoslavia](https://www.icty.org)

## ⚠️ Disclaimer

This website is intended solely for historical documentation and human rights awareness. It does not incite hatred or violence toward any group. All information is sourced from reputable academic and journalistic sources.
